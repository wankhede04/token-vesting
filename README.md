# Token Vesting

Vesting contract for equity distribution according to various classes(designation)

## Steps
Install dependencies
```sh
yarn
```

Hardhat help
```sh
npx hardhat help
```

Compile contract(s)
```sh
yarn complie
```

Run tests
```sh
yarn test
```

Deploy contracts
```sh
NETWORK=<network> yarn deploy
```

Remove artifacts
```sh
yarn clean
```
